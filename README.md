## 自己的收集使用教程具体找作者
 ### fork 好后，点击 Settings 、 secrets 、 New repository secrets 进行添加相关数据
 参数如下：  
|  Name | Value  |
|  --- | --- |
| ENABLE_UNICOM  | 直接填写 true |
| UNICOM_APPID  | 填写获取的 appid |
| UNICOM_USER  | 手机号 |
| UNICOM_PASSWORD  | 服务密码 |   

#### 添加以上数据之后，进入 action ，点击 star 开始运行   
#### 如果未运行，随便找一个文件修改，添加一个空格即可  
#### 并发数我改成2了。
